Echo Framework

Echo is a web framework for building scalable and high-performance web applications in the Go programming language


Echo Demo

Echo Installation

$ go get github.com/labstack/echo/v4

Basic Usage

import 'github.com/labstack/echo/v4'

func main(){
  e:=echo.New()
  e.Logger.Fatal(e.Start(":8080"))
}


Middlewares

Importing echo's middleware

import 'github.com/labstack/echo/v4/middleware'

  • CORS

e.Use(middleware.CORS())

  • Rate limiter

// Request limitaion: 1 per second
e.Use(middleware.RateLimiter(middleware.NewRateLimiterMemoryStore(1)))


Accessing Data from a request

// URL parameter and application/json
e.GET("/api/end-point", func(c echo.Context) error {
  param_value := c.QueryParam("param_name")
}     

// POST method and application/json
e.POST("/api/end-point", func(c echo.Context) error {
  formValue := c.FormValue("inputName")
}
